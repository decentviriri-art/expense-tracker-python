# Expense Tracker

A menu-driven Expense Tracker application built with Python.

## Features

- Add expenses
- View expenses
- Show total expenses
- Show highest expense
- Show lowest expense
- Show average expense

## Technologies Used

- Python 3

## How to Run

Run the program using:

```bash
python expense_tracker.py
```
